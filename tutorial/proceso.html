# Pasos para su realización

# Sin menú customizado

Crea el componente que desees heredado de una ya existente que resulte similar.

Apartir de esto, haremos que el componente implemente Serializable, para así poder guardar los cambios que se realicen a estos componentes.

Crearemos todos los elementos que estemos interesados en implementar, y para que le código pueda acceder a ellos tienen que tener sus respectivos getters y setters, estos públicos.

# (Se recomienda el uso de getters y setters generados automáticamente).
public class PersonalLabel extends JLabel implements Serializable {
    private boolean ComponentePersonalizado;
    public boolean isComponentePersonalizado(){
        return ComponentePersonalizado;
    }
    public void setComponentePersonalizado(boolean ComponentePersonalizado){
        this.ComponentePersonalizado = ComponentePersonalizado;
    }
}

Después de esto guardaremos y compilaremos el componente personalizado, para luego añadirlo a la paleta de componentes gráficos, desde la cual lo podremos mover a la aplicación.

Una vez hecho todo esto, desde el Properties del componente podremos acceder a los elementos que implementamos en el componente personalizados, y cualquier cambio que realicemos a estos se guardará.

# (Para que estos cambios realicen acciones, tendremos que implementar acciones a estos elementos)

# Con menú customizado

Crear un componente con un menú customizado es algo más complejo que crearlo sin él, dado que para hacer esto debemos de realizar muchos más pasos y modificar mucho más código.

Iniciaremos creando un componente el cual extenderá de uno ya existente, e implementará Serializable, para que todos sus datos se guarden.

Podremos usar elementos ya existentes que deseemos modificar con un panel personalizado, pero en caso de desear usar elementos también personalizados deberemos crearlos, estos siempre extendiendo a Serializable.

En caso de realizar estos elementos personalizados deberemos de crearles un constructor, el cual iniciará todos los elementos que este contenga, y realizará todos los procesos requeridos para la creación del elemento. Posteriormente generaremos los getters y setters de todos los elementos que este elemento personalizado contenga.

# (Siempre se recomienda el uso de la generación automática de código de NetBeans para crear getters y setters)

En el componente principal haremos lo mismo y crearemos getters y setters para todos los elementos que deseemos añadirle, y los iniciaremos con la creación del componente.

Crearemos ahora un JPanel en el cual añadiremos los componentes que deseemos de forma que estos nos permitan customizar los elementos personalizados creados previamente. La mayoría de elementos ya existentes tienen sus propios Chooser, asi que podemos apoyarnos en estos para añadir nuevas funciones que modifiquen los subelementos ya existentes.

Aparte de esto tendremos que crear un campo como el siguiente:

public ElementoPersonalizado getSelectedValue() {
    return new ElementoPersonalizado(subelemento1, subelemento2);
}

Después de realizar esto podremos pasar a realizar un ElementoPersonalizadoPropertyEditorSupport.java, el cual contendrá los siguiente elementos.

public class ElementoPersonalizadoPropertyEditorSupport extends PropertyEditorSupport {

    private PanelPersonalizado panelPersonalizadoPanel = new PanelPersonalizadoPanel();

    @Override
    public boolean supportsCustomEditor() {
        return true;
    }

    @Override
    public Component getCustomEditor() {
        return panelPersonalizadoPanel;
    }

    @Override
    public String getJavaInitializationString() {
        ElementoPersonalizado elementoPersonalizado = elementoPersonalizadoPanel.getSelectedValue();
        return "(customStringToCreateNewItem)";
    }

    @Override
    public Object getValue() {
        return coresPanel.getSelectedValue();
    }
}
